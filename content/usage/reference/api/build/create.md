## Endpoint

```
/docs/coming/{soon}
```

| Param | Description |
|---|---|


`{soon}` - documentation coming soon

## Permissions

Documentation Coming Soon!

## Response codes

| Status Code | Description |
|---|---|
| 200 | Everything looks good! |

## Example Response Body

```json
{
  "docs": "Coming soon"
}
```
